# Audio Visualizer
An app that visualize the audio playing from your computer.

Build w/ C++, [SDL3 + imgui + SDLrenderer](https://github.com/ocornut/imgui/tree/master/examples/example_sdl3_sdlrenderer3), [kissfft](https://github.com/mborgerding/kissfft), [miniaudio](https://github.com/mackron/miniaudio)
